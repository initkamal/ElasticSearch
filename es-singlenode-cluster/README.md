Elasticsearch Single Node Cluster
=========

Using: 
        Terraform
        Ansible

Requirements
------------

Terraform v0.15.4
ansible 2.9.6
AWS Access Key and AWS Secret Key

Dependencies
------------

Export [ Environment Variables ]

        $export AWS_ACCESS_KEY_ID="anaccesskey"
        $export AWS_SECRET_ACCESS_KEY="asecretkey"


Example Plan
----------------

        $terraform init
        $terraform plan
        $terraform apply
